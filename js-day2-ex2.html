Event bubbling is the order of how events are "bubbled" in the DOM tree. When you run an event, it will move through the dom starting from the parent elements in a hierarchical order.
This lets you have multiple child elements that share an event handler without having to duplicate it.
In my solution here, I wrap all the divs in a parent div called "outer", which lets all the child divs get access to the eventhandler in my script tag.
